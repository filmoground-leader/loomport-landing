## Tech Stack

- **Frontend**: React 18 + Tailwind CSS
- **Icons**: Lucide React
- **Animations**: CSS & React state management
- **Backend**: Google Apps Script + Google Sheets
- **Hosting**: Vercel
- **Domain**: loomport.com

## Getting Started

### Prerequisites

- Node.js 16+
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
